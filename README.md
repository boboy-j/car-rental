# 🚗 鼎盛汽车租赁 - H5 展示小程序

一个功能完整的汽车租赁 H5 展示小程序，采用微信小程序风格设计，支持数据持久化存储，无需后端服务即可运行。

## ✨ 功能特性

### 📱 首页展示
- 公司品牌展示（名称、标语）
- 图片轮播广告位
- "关于我们" 公司简介
- 四大服务优势展示
- 热门车型推荐
- 联系方式（电话一键拨打）

### 🚙 全部车型
- 按品牌、类型、价格范围筛选
- 车型卡片展示（图片、名称、标签、价格）
- 点击查看车辆详情

### 📋 车辆详情
- 大图展示
- 详细参数信息（座位、品牌、类型）
- 车辆详细介绍
- 租赁说明
- 一键电话咨询

### ⚙️ 管理后台（数据持久化）
- **公司信息管理**：名称、标语、电话、地址、简介、轮播图
- **车辆管理**：添加、编辑、删除车辆
  - 车辆名称、品牌、类型、座位数、日租价格
  - 车辆图片 URL、描述、租赁说明
- 所有数据保存在浏览器 localStorage，刷新不丢失

### 📱 小程序码
- 自动生成当前页面的二维码
- 支持保存/下载二维码图片
- 用户扫码即可打开小程序

## 🛠️ 技术栈

- **纯原生** HTML5 + CSS3 + JavaScript
- **无框架依赖**，轻量快速
- **二维码** 使用 [qrcodejs](https://github.com/davidshimjs/qrcodejs) 库
- **数据持久化** 浏览器 localStorage
- **Unsplash** 免费图片作为默认配图

## 🚀 使用说明

### 方式一：直接打开
直接用浏览器打开 `index.html` 即可使用。

### 方式二：部署到服务器
将所有文件上传到 Web 服务器（Nginx / Apache / 静态托管等），通过域名访问。

### 方式三：微信小程序内嵌
可将此 H5 页面嵌入微信小程序 WebView 中运行。

## 🔧 自定义配置

1. 打开 `index.html` → 进入 **管理后台** → 编辑公司信息和车辆数据
2. 修改默认数据：编辑 `js/data.js` 中的 `DEFAULT_COMPANY` 和 `DEFAULT_CARS`
3. 修改主题色：编辑 `css/style.css` 中的 `--primary` 变量
